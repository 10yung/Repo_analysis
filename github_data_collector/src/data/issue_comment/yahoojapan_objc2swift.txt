In Swift 3.0, C-style for-loops will be deprecated, so instead of this:

``` swift
// Generated by XCSwiftr
for var i = 1; i <= 10; i++ {
    print("\(i)")
}
```

It should be the swift-style for-loop:

``` swift
for i in 0...10 {
     print("\(i)")
}
```
