<h2>3.8.2</h2>
<p><em>Released 01/10/2020</em></p>
<p><strong>Bugfixes:</strong></p>
<ul>
<li>We fixed a regression in <a href="https://on.cypress.io/changelog-3-8-0">3.8.0</a> on Linux when <code>unzip</code> is not installed where the progress of <code>cypress install</code> would show up to 200%. Fixes <a href="https://github-redirect.dependabot.com/cypress-io/cypress/issues/5967">#5967</a></li>
<li>Child elements of an element that uses both <code>transform</code> and <code>height</code> or <code>width</code> are now properly seen as visible during visiblity checks. Addresses <a href="https://github-redirect.dependabot.com/cypress-io/cypress/issues/5974">#5974</a>.</li>
<li>We now properly check backface visibility when the parents of a target element have the CSS style <code>transform-style: preserve-3d</code>. Fixes <a href="https://github-redirect.dependabot.com/cypress-io/cypress/issues/5682">#5682</a>.</li>
<li>DOM elements with a width or height of 0 and <code>overflow: hidden</code> are now properly calculated as invisible during visibility checks. Fixes <a href="https://github-redirect.dependabot.com/cypress-io/cypress/issues/6093">#6093</a>.</li>
<li>We fixed a regression in <a href="https://on.cypress.io/changelog-3-7-0">3.7.0</a> where using <a href="https://on.cypress.io/type"><code>.type()</code></a> on number inputs to type non-number characters would not type the correct value. Fixes <a href="https://github-redirect.dependabot.com/cypress-io/cypress/issues/6055">#6055</a>.</li>
</ul>
<p><strong>Misc:</strong></p>
<ul>
<li>Messages from <a href="https://on.cypress.io/log"><code>cy.log()</code></a> are now truncated at 50 lines the Command Log instead of the 1 line. Addresses <a href="https://github-redirect.dependabot.com/cypress-io/cypress/issues/5625">#5625</a>.</li>
<li>We now capture GitLab CI's <code>CI_JOB_NAME</code> environment variable. Addressed in <a href="https://github-redirect.dependabot.com/cypress-io/cypress/pull/6111">#6111</a>.</li>
<li>Types for <a href="https://on.cypress.io/catalog-of-events#Binding-to-Events">event emitters</a> <code>removeListener</code> and <code>removeAllListeners</code> have been added to <code>Cypress</code> and <code>cy</code>. Addresses <a href="https://github-redirect.dependabot.com/cypress-io/cypress/issues/5228">#5228</a>.</li>
<li>Types for the <code>runUrl</code> property returned from <code>cypress.run()</code> has been added. Addresses <a href="https://github-redirect.dependabot.com/cypress-io/cypress/issues/6087">#6087</a>.</li>
<li>We're continuing to make progress in converting our codebase from CoffeeScript to JavaScript. Addresses <a href="https://github-redirect.dependabot.com/cypress-io/cypress/issues/2690">#2690</a> in <a href="https://github-redirect.dependabot.com/cypress-io/cypress/pull/5947">#5947</a> and <a href="https://github-redirect.dependabot.com/cypress-io/cypress/pull/6075">#6075</a>.</li>
</ul>
<p><strong>Dependency Updates:</strong></p>
<ul>
<li>Upgraded <code>ansi_up</code> from <code>4.0.3</code> to <code>4.0.4</code>. Addressed in <a href="https://github-redirect.dependabot.com/cypress-io/cypress/pull/6074">#6074</a>.</li>
<li>Upgraded <code>electron</code> from <code>7.1.4</code> to <code>7.1.7</code>. Addressed in <a href="https://github-redirect.dependabot.com/cypress-io/cypress/pull/6085">#6085</a>.</li>
</ul>
</blockquote>
