The above program demonstrate the following operations on doubly linkedlist data structure.
1. creation of linked list
2. traversing of linked list(both the direction)
3. Inserting a node at begining of list
4. inserting a node at end of list.
5. Inserting a node after a given node
6. Inserting a node before a given node
7. reversing a doubly linked list.
8. Finding pairs with given sum
9. insert node in sorted way.
10. calculating tiplet with sum equal to value.
11. deleting all occurances of given node.
12. removing all duplicates from sorted linked list.
