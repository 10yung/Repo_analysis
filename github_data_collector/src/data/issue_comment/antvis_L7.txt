* **L7 Version**: 1.3.0
* **Platform**: windows10

需要在 `CONTRIBUTING.md` 中补充 Windows 环境下开发项目的内容。

1. 由于使用了 Yarn workspace，首先需要安装 Yarn：https://yarnpkg.com/en/docs/install#windows-stable
2. `yarn install` 安装依赖，完成 workspace 链接
3. [测试方案](https://github.com/antvis/L7/blob/master/dev-docs/%E8%87%AA%E5%8A%A8%E5%8C%96%E6%B5%8B%E8%AF%95%E6%96%B9%E6%A1%88.md)依赖 [headless-gl](https://github.com/stackgl/headless-gl#windows)，因此需要 `node-gyp` 编译本地依赖，[这一步](https://github.com/nodejs/node-gyp#on-windows)是 Windows 下出现问题最多的。
  a. 首先以管理员身份启动 PowerShell

<img width="408" alt="屏幕快照 2019-11-28 下午3 59 13" src="https://user-images.githubusercontent.com/3608471/69788332-3b70d280-11f9-11ea-839d-7da4553ea1f1.png">

  b. 运行 `npm install --global --production windows-build-tools`，安装 Microsoft's windows-build-tools 。这一步会安装 Python 2.7

<img width="529" alt="屏幕快照 2019-11-28 下午4 49 47" src="https://user-images.githubusercontent.com/3608471/69791299-2bf48800-11ff-11ea-85d3-dc6dcba968e5.png">

  c. [参考](https://github.com/imagemin/imagemin-pngquant/issues/46#issuecomment-515808859)，需要下载 Visual C++ Build Tools 并安装。Win7 以下安装 [2013](https://www.microsoft.com/en-us/download/details.aspx?id=40760)，Win10 及以上安装 [2015](https://www.microsoft.com/en-us/download/details.aspx?id=48159 )
  d. 重启
