This PR is just an idea for implementation of issue #7,  which originally refers to `<embed>` instead of `<object>`.
As I mentioned in issue #7, target attribute in `<a>` can have _top, _parent  or _blank.
So I added a select box named "Link target" so that users can choose which value is set to the target attribute.
