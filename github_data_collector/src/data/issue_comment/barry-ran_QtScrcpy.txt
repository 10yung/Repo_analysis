建议将投屏窗口旁工具栏上的按钮做上按键映射
例如: Ctrl + Enter 桌面键
有两个用处, 一是习惯用键盘操作的可以在键盘上执行返回, 返回桌面, 多任务切换 等操作, 不用费劲动用鼠标去找那个很小的按钮. 
第二个用处是, 配合电脑上的鼠标手势软件映射快捷键, 例如用WGesture做个例子: 定义在投屏界面上时按住鼠标右键上移映射 Ctrl + Enter , 就可以借助鼠标手势完成回到桌面的操作. 
这个软件可能还有一个使用场景, 是无线投屏上, 直接一个屏幕处理所有工作, 手机仍一边, 所以可能手机屏幕离电脑屏幕2米远, 这时要查看手机内容就要解锁
解锁就麻烦了, 要么起身过去指纹解锁, 要么在电脑上电源键, 上滑, 点击输入解锁密码
但其实可以三行shell解决:
shell input keyevent 26
shell input swipe 540 1800 540 200 1000
shell input text <password>
如果能做个按键, 一键执行多行shell命令, 就可以做到一键开锁的目的.
