个人理解。理论上，get、post、put的功能确实可以互通，但是约定上用法不同，导致服务器上的方法一般写得也会不同（自己硬要写个能互通的方法也可以，但不符合一般的习惯），使得实际使用时，就是不一样的。完全就是大部分地方解释的，GET获得数据，POST提交、创建数据、PUT修改数据。
